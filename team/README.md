## How the task links stay accurate
Each profile's task table is generated by scanning every file under
`02-missions/**/tasks/*.md` (plus `02-missions/convergence-synthesis.md`)
for its `owner:` frontmatter field — the same "GitHub already surfaces
this, don't duplicate it in a folder" principle used elsewhere in this repo
(see root `README.md`). If task ownership changes, the task file is the
source of truth; re-run the scan to refresh these tables.
